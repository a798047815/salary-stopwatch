## 🚀 使用方法
1. 直接打开 `index.html` 即可使用，无需任何依赖
2. 点击「⚙️ 设置」按钮配置你的薪资和工作时间
3. 点击「▶️ 开始赚钱」启动计时器（工作时间内会自动启动）
4. 摸鱼时点击底部「🎮 摸鱼」tab畅玩三款经典小游戏
5. 想跳槽时点击「💼 工作」tab查看最新高薪岗位推荐

## 🎮 游戏操作说明
| 游戏 | 操作方式 |
|------|----------|
| 贪吃蛇 | 方向键 ↑ ↓ ← → 控制方向 |
| 飞机大战 | 方向键 ← → 移动，空格键发射子弹 |
| 俄罗斯方块 | 方向键 ← → 移动，↑ 旋转，↓ 加速，空格快速下落 |

所有游戏支持暂停/继续，最高分永久保存。
