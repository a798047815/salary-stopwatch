# 赚钱秒表 (Salary Stopwatch) 🏆

一个让上班变得有趣的秒表小程序，实时显示你今天赚了多少钱。

## 功能特性

- 💰 **实时赚钱** - 每分每秒看到收入增长
- ⏰ **上下班时间** - 自定义工作时间
- 🍚 **午休设置** - 自动扣除午休时间
- 🎯 **里程碑系统** - 达成目标庆祝
- 📊 **统计报表** - 今日/本周/本月收入
- 🔥 **连续打卡** - 记录连续上班天数
- 🏆 **成就系统** - 解锁各种成就

## 项目结构

## 使用方法

1. 下载 **微信开发者工具**
2. 导入此项目
3. 编译运行

或者改成 H5 版本发布到网页。
