# 赚钱秒表 - 产品需求文档

## 1. 项目概述

**项目名称**：赚钱秒表 (Salary Stopwatch)
**项目类型**：微信小程序 / H5 Web App
**一句话描述**：一个实时显示上班赚钱进度的秒表工具，让工作时间变得有意义
**目标用户**：上班族、打工族

---

## 2. 功能清单

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 工资设置 | 设置每日工资或月薪，按工作天数折算每日工资 | P0 |
| 上下班时间设置 | 设置每日上班时间、下班时间、午休时长 | P0 |
| 赚钱秒表 | 实时累计今日收入，每秒更新 | P0 |
| 休息自动暂停 | 非工作时间（下班后）自动暂停计时 | P0 |
| 收入里程碑 | 达成特定金额时播放庆祝动画 | P1 |
| 今日/本周/本月统计 | 收入汇总数据展示 | P1 |

### 2.2 增强功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 下班倒计时 | 显示距下班剩余时间 + 预计收入 | P1 |
| 时薪显示 | 实时显示当前时薪 | P2 |
| 连续打卡 | 记录连续上班赚钱天数 | P2 |
| 成就系统 | 达成特定成就解锁徽章 | P2 |
| 语音播报 | 每满一定金额语音提醒（可选） | P2 |
| 主题皮肤 | 不同主题/配色方案 | P3 |
| 数据导出 | 导出收入记录 | P3 |

### 2.3 设置项

- 每日工资 / 月薪
- 上班时间、下班时间
- 午休开始时间、午休时长
- 货币符号（¥/$）
- 里程碑金额设置
- 庆祝动画开关
- 语音播报开关
- 深色/浅色模式

## 4. 交互设计

### 4.1 赚钱动画
- 金额数字使用计数器动画，每秒增加
- 达到里程碑时：彩纸特效 + 轻微震动 + 音效
- 数字增加时有放大+绿色高亮效果

### 4.2 下班倒计时
- 实时更新，精确到秒
- 配合预计收入显示

### 4.3 成就系统
- 达成条件自动解锁
- 徽章图标 + 成就名称 + 获得日期

---

## 5. 技术方案

### 5.1 技术栈
- **微信小程序**：原生开发 / Uni-app
- **H5版本**：React + Vite
- **数据存储**：LocalStorage（小程序用 wx.setStorage）

### 5.2 核心算法

```javascript
// 每秒收入计算
每秒收入 = 每日工资 / 每日工作秒数

// 每日工作秒数
工作秒数 = (下班时间 - 上班时间) - 午休秒数

// 里程碑检测
if (当前收入 >= 里程碑 && !已达成) {
  触发庆祝动画
}
```

## 7. 成就预设

| 成就名称 | 达成条件 |
|----------|----------|
| 首次开工 | 首次使用 |
| 全勤一周 | 连续7天使用 |
| 月入过万 | 单月收入超过10000 |
| 早起鸟儿 | 8点前开始工作 |
| 夜猫子 | 20点后还在工作 |
| 连续打卡 | 连续30天使用 |

## 8. 竞品分析

暂无直接竞品，类比产品：
- 时间追踪类 app（如 Toggl）
- 工资计算器
- 习惯打卡类 app

**差异化点**：将时间与金钱直观关联，创造"玩游戏打怪升级"的赚钱体验
